# OCR_GIF
This is a sample project for extracting texts from a GIF.


STEPS: 

1. ```shell
pip install virtualenv
```
2. ```shell
virtualenv ORCtesse
```

3.
> For MacOS and Linux
 ```shell
source ORCtesse/bin/activate
```
> For Windows

```shell
ORCtesse\Scripts\activate
```

4. ```shell
pip install -r requirements.txt
```
